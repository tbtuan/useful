## Class

```php
class Point {

	private $x;
	private $y;

	// __contruct = constructor
	function __construct($x, $y) {
		// this isn't optional
		$this->x = $x;
		$this->y = $y;
	}
	// Getter
	function getX() {
		return $this->x;
	}

	function getY() {
		return $this->y;
	}

	// Function with $array (passed by ref) default parameters
	function &returnFirstElement(&$array, $a=123) {
	// & before function name + $array
	  return $array[0];
	}

}

$p = new Point(1, 2);
```
