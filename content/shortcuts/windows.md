## Windows Explorer

| Description                | Shortcut                                          |
| -------------------------- | ------------------------------------------------- |
| Open Windows Explorer      | <kbd>Win</kbd> + <kbd>E</kbd>                     |
| New folder                 | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>N</kbd> |
| Jump to the navigation bar | <kbd>Ctrl</kbd> + <kbd>L</kbd>                    |
