## grep


| Description                                                                          | Command                              |
| ------------------------------------------------------------------------------------ | ------------------------------------ |
| Search "pattern" in "files" | `grep -n [TERM] [FILES]` |
| Concatenate (cat) and search | `cat [FILES] | grep [TERM]` |
